# Running Code

MAKE SURE YOU HAVE yarn INSTALLED ON YOUR OS

1. Open the code directory folder in your VSCode
2. Go to 'Terminal' and run this code

```

yarn dev
```

3. Open [[localhost:3000/petpal](http://localhost:3000/petpal)] on your browser
